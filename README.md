# AiRecruiter-
AIRecruiter: цифровой продукт для интеллектуального подбора персонала в консалтинге
Gрототип системы интеллектуального скрининга резюме для консалтинговых и аналитических позиций. Проект реализован в рамках курсовой работы и сочетает анализ реальных резюме, построение моделей машинного обучения и разработку веб‑интерфейса для рекрутера.

1. Цели проекта
Сформировать датасет резюме кандидатов на позиции в консалтинге и провести его описательный анализ.
Построить и оценить модели, предсказывающие вероятность получения оффера на основе признаков, извлечённых из резюме.
Разработать прототип веб‑приложения, поддерживающего скоринг кандидатов, кластеризацию профилей и генерацию отчёта для рекрутера.

2. Основной функционал прототипа
Прототип AIRecruiter реализован как веб‑приложение с архитектурой клиент–сервер. Основные возможности:

Загрузка резюме в форматах PDF и DOCX.

Автоматический парсинг текста и извлечение признаков (28 признаков по шести группам компетенций).

Расчёт вероятности получения оффера с помощью обученной модели (LightGBM + ансамблевый регрессор).

Отнесение кандидата к одному из кластеров профиля (сегментация K‑Prototypes).

Отображение скор‑карты кандидата и генерация отчёта для рекрутера.

4. Технологический стек
Язык: Python 3.x.

Аналитика и ML: pandas, scikit-learn, xgboost, lightgbm, optuna.


Парсинг резюме: pdfplumber, PyMuPDF, pytesseract 

3. Ограничения и статус проекта
Прототип работает на ограниченном датасете (597 резюме одной компании), что сдерживает качество моделей и переносимость выводов.

Модели демонстрируют умеренный уровень предсказательной способности и предназначены для вспомогательного скоринга, а не для автономного принятия кадровых решений.

Текущая версия реализует MVP: обработку отдельных резюме, базовую визуализацию скоринга и кластеризацию профилей.

7. План развития (roadmap)
Добавление пакетной обработки резюме и интеграции по API с ATS‑системами.

Расширение словарей признаков с учётом новых специализаций (data engineering, ESG‑консалтинг и др.).

Внедрение семантического поиска похожих кандидатов на основе Sentence‑BERT.

Расширение языковой поддержки и подключение модулей для аналитики рекрутинговой воронки.
